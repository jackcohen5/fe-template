## Features

* React 16
* Webpack 4
* Babel 7
* ESLint
* Hot Module Replacement

## Getting Started

### Install app dependencies

You can install Yarn [directly](https://yarnpkg.com/en/docs/install) or with [yvm](https://github.com/tophat/yvm)

* `yarn install`

### Running the app

* `yarn start`
* visit `http://localhost:3000/`
